01 — Project Overview

Problem & Solution

Bangladesh is home to an extraordinarily rich tapestry of regional dialects that differ substantially in phonology, vocabulary, and prosody from standard literary Bangla. Mainstream ASR systems consistently fail on dialectal speech, leaving millions of speakers underserved. SHOBDOTORI tackles this gap directly by fine-tuning OpenAI's Whisper Medium on a curated 20-dialect regional dataset, enabling robust, accurate transcription of dialectal audio into standardized written Bangla.

02 — System Architecture

Model Pipeline

🎙️ Audio Input
WAV file · 16 kHz mono · any regional dialect
↓
📊 Whisper Feature Extractor
Log-Mel Spectrogram · 80 mel bins · 30s window
↓
🧠 Whisper Medium Encoder
~307M parameters · Transformer · contextual embeddings
↓
📝 Whisper Decoder
Autoregressive generation · Bengali language head · task=transcribe
↓
✅ Standard Bangla Output
Unicode Bengali text · standardized orthography

Base Model Chain

  • 1️⃣
    openai/whisper-medium
    OpenAI's original multilingual model (300M params). Foundation for Bengali speech.
  • 2️⃣
    bengaliAI/tugstugi-regional-asr
    Fine-tuned on BEN10 Kaggle competition data. Used as our training starting point.
  • 3️⃣
    BITWISEMIND Fine-tune
    Further fine-tuned on our custom 20-dialect regional dataset. Final model.

Key Technical Choices

  • ⚡
    FP16 Mixed Precision — halved GPU memory while maintaining accuracy
  • 🔀
    Gradient Checkpointing — enabled training the full 300M model on T4
  • 📦
    Gradient Accumulation ×4 — effective batch size of 16 from 4 per device
  • 📏
    WER Evaluation — checkpoint selected by lowest Word Error Rate

06 — Inference Summary

Using the Fine-Tuned Model

Python — Quick Inference
from transformers import (
    WhisperForConditionalGeneration,
    WhisperProcessor
)
import librosa, torch

# Load fine-tuned model
path = "./whisper-medium-bengali-regional-final"
processor = WhisperProcessor.from_pretrained(path)
model = WhisperForConditionalGeneration.from_pretrained(path)
model.eval()

# Load audio (any Bangla dialect)
audio, _ = librosa.load("dialect_audio.wav", sr=16000)

# Extract features & transcribe
inputs = processor(audio, sampling_rate=16000,
                   return_tensors="pt")
with torch.no_grad():
    ids = model.generate(inputs.input_features)

result = processor.batch_decode(ids,
             skip_special_tokens=True)[0]
print(f"Transcription: {result}")
